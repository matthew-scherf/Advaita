I. FORMAL SIGNATURE

--------------------------------------------------------------------------------

A. SORTS (Types)

--------------------------------------------------------------------------------

 

Obj       -- Objects (entities): Brahman, jīvas, phenomena, sheaths, ego

Level     -- Reality levels: Param, Vyav, Prat

Time      -- Temporal points (discrete linear order)

Event     -- Events (reified occurrences)

 

Level Constants:

  Param : Level    -- Paramārthika (ultimate reality)

  Vyav  : Level    -- Vyāvahārika (conventional reality)

  Prat  : Level    -- Prātibhāsika (illusory appearance)

 

--------------------------------------------------------------------------------

B. PREDICATES ON OBJECTS

--------------------------------------------------------------------------------

 

Metaphysical Status:

  A(x)              -- x is Absolute (Brahman/Ātman)

  C(x)              -- x is Conditioned (phenomenal, dependent)

  Y(x)              -- x is You (ultimate subject/witness, Ātman)

 

Phenomenal Properties:

  T_p(x)            -- x is Temporal

  S_p(x)            -- x is Spatial

  Q_p(x)            -- x has Qualities

  Phi(x)            -- x has phenomenal properties (T_p ∨ S_p ∨ Q_p)

 

Consciousness Properties (Saccidānanda):

  Sat(x)            -- x has being-nature (pure existence)

  Cit(x)            -- x has consciousness-nature (pure awareness)

  Ananda(x)         -- x has bliss-nature (pure fullness)

  Saccidananda(x)   -- Sat(x) ∧ Cit(x) ∧ Ananda(x)

 

Reality Level:

  Level(x,ℓ)        -- x exists at reality level ℓ

 

Ontological Relations:

  Cond(x,y)         -- x ontologically grounds y

  MayaPow(a,x)      -- Māyā-power from a manifests as x

  MayaLevel(r,ℓ)    -- Māyā-relation r operates at level ℓ

  Superimposed(x,y) -- x superimposed on y (adhyāsa)

  Appears(x,y)      -- x appears as y (vivarta)

  IgnoranceOf(s,x)  -- Subject s is ignorant of x's true nature (avidyā)

  Upadhi(u,x)       -- u is limiting adjunct for x

  Sublates(k₁,k₂)    -- Knowledge k₁ cancels knowledge k₂ (bādha)

 

Individual Consciousness:

  Jiva(x)           -- x is individual self (jīva)

  Isvara(x)         -- x is personal God (Īśvara)

  Embodied(j,b)     -- Jīva j embodied in body b

 

Awareness Relations:

  Perceives(s,o)    -- Subject s perceives object o (dualistic)

  Witnesses(w,z)    -- w witnesses z (non-dualistic, sākṣin)

  Possesses(s,k)    -- Subject s possesses/has realized knowledge k

 

Knowledge Structure:

  Knower(x)         -- x is knower (pramātṛ)

  Known(x)          -- x is known (prameya)

  Knowing(x)        -- x is knowing-process (pramāṇa)

  DistinctAspects(k,n,g) -- k,n,g appear as distinct aspects

  Knowledge(k)      -- k is cognition (jñāna)

  Liberating(k)     -- k is liberating knowledge (mokṣa-jñāna)

 

Change and Transformation:

  RealChange(x,y)   -- x really transforms into y (pariṇāma)

  Changes(x)        -- x undergoes change (vikāra)

  Born(x)           -- x came into being (jāta)

  Dies(x)           -- x ceases to be (mṛta)

 

Five Sheaths (Pañca-kośa):

  Annamaya(x)       -- Physical sheath (food body)

  Pranamaya(x)      -- Vital sheath (prāṇa)

  Manomaya(x)       -- Mental sheath (manas)

  Vijnanamaya(x)    -- Intellectual sheath (buddhi)

  Anandamaya(x)     -- Bliss sheath (ānanda reflection)

  Sheath(x)         -- x is any sheath (disjunction of above)

  Layer(x,y)        -- x is outer layer of y

 

Three Guṇas:

  Sattva(x)         -- x exhibits sattva (purity, light)

  Rajas(x)          -- x exhibits rajas (activity, passion)

  Tamas(x)          -- x exhibits tamas (inertia, darkness)

  HasGuna(x)        -- x has at least one guṇa

 

Ego Structure:

  Ego(x)            -- x is ego (ahaṃkāra, sense of "I")

  ApparentSubject(x)-- x appears as subject (false "I")

  Body(x)           -- x is body-mind complex

  Identifies(x,y)   -- x identifies with y

 

Spacetime:

  SpaceItself(x)    -- x is space as entity (ākāśa)

  TimeItself(x)     -- x is time as entity (kāla)

 

Existence:

  EE(e)             -- e exists (event)

 

--------------------------------------------------------------------------------

C. PREDICATES ON TIME AND EVENTS

--------------------------------------------------------------------------------

 

Temporal Relations:

  Before(t₁,t₂)     -- Temporal point t₁ precedes t₂

 

Event Relations:

  OccursAt(e,t)     -- Event e occurs at time t

  EventOf(e,x)      -- Event e involves object x

  CausesEvent(e₁,e₂) -- Event e₁ causes event e₂

 

Event Constructors:

  PerceptionEvent(s,o) : Event  -- Perception event

  ChangeEvent(x) : Event       -- Change event

  BirthEvent(x) : Event        -- Birth event

  DeathEvent(x) : Event        -- Death event

  -- Phenomenal property (has temporal, spatial, or qualitative aspects)

  -- Being-consciousness-bliss nature

 

Sheath(x) ≡ Annamaya(x) ∨ Pranamaya(x) ∨ Manomaya(x) ∨

            Vijnanamaya(x) ∨ Anandamaya(x)

  -- Any of the five sheaths

 

HasGuna(x) ≡ Sattva(x) ∨ Rajas(x) ∨ Tamas(x)

  -- Has at least one of the three guṇas

 

Brahman ≡ ι a. A(a)

  -- The unique Absolute (definite description)

 

Atman ≡ ι u. Y(u)

  -- The unique Subject (definite description)

III. CORE AXIOMS (A1-A15)

================================================================================

 

The essential metaphysical structure of Advaita Vedanta.

 

A1: Universal Classification

  (a) ∀x. A(x) ∨ C(x)

  (b) ∀x. ¬(A(x) ∧ C(x))

 

A2: Uniqueness of the Absolute

  ∃! a. A(a)


  A2b: ∀a. A(a) → (∀x. x ≠ a → ¬A(x))

  A2b': ∀x. A(x) → x = Brahman

 

A3: Uniqueness of the Subject

  ∃! u. Y(u)


  A3b: ∀u. Y(u) → (∀x. x ≠ u → ¬Y(x))

  A3b': ∀x. Y(x) → x = Atman

 

A4: Subject-Absolute Identity (Tat Tvam Asi Core)

  ∀x. Y(x) ↔ A(x)

 

A5: Self-Grounding of the Absolute

  ∀a. A(a) → Cond(a,a)

 

A6: Universal Grounding

  ∀x. (∃a. A(a) ∧ Cond(a,x))

 

A7: Transcendence of Phenomenality

  ∀a. A(a) → ¬Phi(a)


  A7a: ∀a. A(a) → (¬T_p(a) ∧ ¬S_p(a) ∧ ¬Q_p(a))

 

A7b: Saccidānanda Nature

  ∀a. A(a) → Saccidananda(a)

 

A8: Phenomenality of the Conditioned

  ∀x. C(x) → Phi(x)

 

A9: Asymmetry of Grounding (Non-Absolute)

  ∀x y. Cond(x,y) ∧ Cond(y,x) → (x = y ∧ A(x))

 

A10: Transitivity of Grounding

  ∀x y z. Cond(x,y) ∧ Cond(y,z) → Cond(x,z)

 

A11: Absolute Witnesses Everything

  ∀a x. A(a) → Witnesses(a,x)

 

A12: Self-Luminosity

  ∀a. A(a) → Witnesses(a,a)

 

A13: Subject Does Not Perceive

  ∀u o. Y(u) → ¬Perceives(u,o)

 

A14: Subject is Knower-Known-Knowing

  ∀u. Y(u) → (Knower(u) ∧ Known(u) ∧ Knowing(u))

 

A15: No Real Change in Absolute

  ∀a x. A(a) → ¬RealChange(a,x)

  ∀x y. C(x) ∧ Level(x,Vyav) ∧ Level(y,Prat) → Sublates(x,y)

  ∃x. Level(x,Vyav)

Time structure: discrete linear order.

 

T1: Irreflexivity

  ∀t. ¬Before(t,t)

 

T2: Transitivity

  ∀t₁ t₂ t₃. Before(t₁,t₂) ∧ Before(t₂,t₃) → Before(t₁,t₃)

 

T3: Asymmetry

  ∀t₁ t₂. Before(t₁,t₂) → ¬Before(t₂,t₁)

 

T4: Linearity (Optional)

  ∀t₁ t₂. t₁ ≠ t₂ → (Before(t₁,t₂) ∨ Before(t₂,t₁))

 

T5: Non-Empty Time

  ∃t. True

 

T6: Distinct Times

  ∃t₁ t₂. t₁ ≠ t₂

Events as reified occurrences in time.

 

E1: Event Existence Criterion

  ∀e. EE(e) → (∃! t. OccursAt(e,t))

  ∀e. EE(e) → (∃x. EventOf(e,x))

  ∀x y e_y. Cond(x,y) ∧ EventOf(e_y,y) ∧ EE(e_y) ∧ ¬A(x) →

            (∃e_x. EventOf(e_x,x) ∧ EE(e_x))

================================================================================

 

Consciousness structure: witnessing vs perception.

 

W1: Witnessing is Universal

  ∀w x. Y(w) → Witnesses(w,x)

  ∀w x e. Witnesses(w,x) → ¬(e = PerceptionEvent(w,x))

  ∀s o. Perceives(s,o) → (∃e t. e = PerceptionEvent(s,o) ∧ OccursAt(e,t))

  ∀s. (∃o. Perceives(s,o)) → C(s)

  ∀w. (∃x. Witnesses(w,x) ∧ x ≠ w) → Y(w)

  ∀u. Y(u) → (∀k n g. DistinctAspects(k,n,g) →

  ∀x. C(x) → (∃k. Knowledge(k) →

            (∃k' n g. DistinctAspects(k',n,g) ∧

W9: Liberating Knowledge Removes Ignorance

  ∀k j a. Liberating(k) ∧ Jiva(j) ∧ A(a) →

  ∀u. Y(u) → ¬(∃k n g. DistinctAspects(k,n,g) ∧

                     (u = k ∨ u = n ∨ u = g) ∧

                     (u ≠ k ∨ u ≠ n ∨ u ≠ g))

 

W11: Witness-Absolute Identity

  ∀w. (∀x. Witnesses(w,x)) → A(w)
